# Анализ дебита и кредита

Десктопное приложение для подсчёта дебита, кредита и разницы из Excel-файлов (.xlsx).

## Установка (один раз)

1. Убедитесь, что установлен Python 3.10+ (проверить: `python --version` в терминале)

2. Откройте папку проекта в терминале VS Code (`Terminal → New Terminal`)

3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Запуск

```
python main.py
```

## Как пользоваться

1. Нажмите **Обзор…** и выберите Excel-файл (.xlsx)
2. Нажмите **Рассчитать**
3. В таблице появятся результаты по каждому листу, а наверху — итоговые суммы

## Логика расчёта

- Приложение автоматически **находит строку-заголовок** и определяет,
  в каких столбцах находятся «дебит» и «кредит» (по вхождению слова в название столбца)
- Анализируются **все листы** книги
- Строки, где в пояснении (столбцы правее дебит/кредит) упоминается **вклад**
  (перечисление / возврат средств), **исключаются** из расчёта
- Строки с упоминанием **процентов со вклада** — **не исключаются** (доход остаётся)
